## 8. Synthèse et recommandations prioritaires

### 🔴 Actions critiques

#### 1. Résoudre la cannibalisation www / non-www
- **Impact** : Le trafic et les mots clés sont divisés entre www et non-www dans l'index Google
- **État technique actuel** :
  - ✅ `.htaccess` : redirect 301 www → non-www déjà en place (lignes 12-13)
  - ✅ Balises canonical : toutes les 68 pages pointent déjà vers la version non-www (0 canonical www détecté)
  - ✅ Sitemap : 63 URLs, toutes en version non-www
- **Action restante** (côté Google, pas côté site) :
  - Dans Google Search Console, vérifier que la propriété de domaine est bien consolidée
  - Demander une nouvelle exploration via GSC pour forcer la re-consolidation
  - Vérifier dans GSC > Indexation que les pages www sont bien marquées comme « redirection »
- **Effort** : 1h (vérification GSC uniquement, aucune modification de code nécessaire)

#### 2. Construire un profil de backlinks
- **Impact** : Authority Score = 0, 1 seul backlink → impossible de se positionner sur des requêtes compétitives
- **Action** :
  - Inscription sur les annuaires médicaux (Doctolib, PagesJaunes, Yelp, Google Maps)
  - Demander des liens depuis les sites partenaires (mutuelles, pharmacies locales)
  - Créer du contenu partageable (guides santé, infographies)
  - Viser 10-20 backlinks de qualité dans les 3 prochains mois
- **Effort** : Continu, 2-4h/semaine

#### 3. Diversifier les sources de trafic organique
- **Impact** : 96% du trafic vient de la homepage via 5 mots clés "centre médical"
- **État actuel du contenu** :
  - ✅ psychologie.html : 2640 mots (riche)
  - ✅ kinesitherapie.html : 1881 mots (correct)
  - 🟡 dieteticien.html : 1472 mots (à enrichir légèrement)
  - 🔴 urgence.html : 438 mots (trop court, prioritaire)
  - 🟠 Sous-pages soins-infirmiers : 350-666 mots (à enrichir)
- **Action** :
  - **Priorité 1** : Enrichir `urgence.html` (438 mots → 1500+ mots) — requête "urgence levallois" à pos 14
  - **Priorité 2** : Enrichir les sous-pages soins-infirmiers (pansement-post-chirurgical, retrait-fils-agrafes)
  - **Priorité 3** : Ajouter des FAQ riches avec schema markup sur les pages index spécialités
  - Cibler des mots clés informationnels à fort volume mais faible KD
  - **Note** : 34 pages (psycho + diét. + kiné) viennent d'être déployées (21-22 mars 2026). Attendre 2-4 semaines avant de juger leur performance organique.
- **Effort** : 2-4h pour urgence.html, 2-3h par sous-page soins-infirmiers
